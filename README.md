# UIT Student Captcha

Auto fill word catpcha like this:

```
What color are the leaves of a tree in autumn?(orange) *
What is the second letter in 'hello'?(e) *
```

## Installation

1. [Tampermonkey](https://www.tampermonkey.net/)
2. Install [script](https://github.com/KevinNitroG/uit-student-captcha/releases/latest/download/uit-student-captcha.user.js)

## Usage

1. Go to [UIT Student Portal](https://student.uit.edu.vn/)
2. It will auto fill the captcha
